# Rossi SiteGuard Monitor — Landing v2.0

React + Vite + TypeScript + Tailwind CSS + shadcn/ui

## Build
```bash
npm install
npm run build
```

Output: `dist/` — deploy to `/var/www/siteguard-v2/` on server, served at `/v2`
